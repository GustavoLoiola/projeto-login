# projeto-login
Projeto tela de login

Projeto com foco em usar responsividade na prática HTML5 e CSS3. O projeto é o FrontEnd de uma tela de login adaptada para telas grandes, médias e pequenas com uso de FlexBox e efeito de transição de acordo com a mudança do tamanho de tela.
